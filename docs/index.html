Usage Examples

Import the Library

const amlich = require('amlich');

Convert Solar Date to Lunar Date

Convert a Gregorian date to Vietnamese lunar date. The timezone parameter is UTC offset (7.0 for Vietnam, UTC+7).

// Convert January 1, 2024 to lunar date
const lunarDate = amlich.convertSolar2Lunar(1, 1, 2024, 7.0);
// Returns: [day, month, year, leapMonth]
// Example output: [20, 11, 2023, 0]
// Means: 20th day of 11th lunar month, year 2023

console.log(`Lunar date: ${lunarDate[0]}/${lunarDate[1]}/${lunarDate[2]}`);
console.log(`Is leap month: ${lunarDate[3] === 1 ? 'Yes' : 'No'}`);

Convert Lunar Date to Solar Date

Convert a Vietnamese lunar date back to Gregorian date. Set leapMonth to 1 if the lunar month is a leap month, otherwise 0.

// Convert 1st day of 1st lunar month, year 2024
const solarDate = amlich.convertLunar2Solar(1, 1, 2024, 0, 7.0);
// Returns: [day, month, year]
// Example output: [10, 2, 2024]
// Means: February 10, 2024

console.log(`Solar date: ${solarDate[0]}/${solarDate[1]}/${solarDate[2]}`);

Calculate Julian Day Number

Calculate the Julian day number for a given date. This is the number of days since January 1, 4713 BC (Julian calendar).

// Calculate Julian day for January 1, 2024
const jd = amlich.jdFromDate(1, 1, 2024);
console.log(`Julian day: ${jd}`);
// Output: 2460310

Convert Julian Day to Date

Convert a Julian day number back to a Gregorian date.

// Convert Julian day to date
const date = amlich.jdToDate(2460310);
console.log(`Date: ${date[0]}/${date[1]}/${date[2]}`);
// Output: Date: 1/1/2024

API Reference

jdFromDate(dd, mm, yyyy)

Calculate Julian day number (the number of days between 1/1/4713 BC Julian calendar and dd/mm/yyyy)

Parameters:
  • dd - Day of month (1-31)
  • mm - Month (1-12)
  • yyyy - Year
Returns: Integer Julian day number

jdToDate(jd)

Convert a Julian day number to day/month/year

Parameters:
  • jd - Julian day number (integer)
Returns: Array [day, month, year]

convertSolar2Lunar(dd, mm, yyyy, timeZone)

Convert solar date dd/mm/yyyy to the corresponding lunar date

Parameters:
  • dd - Day of month (1-31)
  • mm - Month (1-12)
  • yyyy - Year
  • timeZone - UTC offset (e.g., 7.0 for UTC+7)
Returns: Array [lunarDay, lunarMonth, lunarYear, leapMonth]

convertLunar2Solar(lunarDay, lunarMonth, lunarYear, lunarLeap, timeZone)

Convert a lunar date to the corresponding solar date

Parameters:
  • lunarDay - Lunar day (1-30)
  • lunarMonth - Lunar month (1-12)
  • lunarYear - Lunar year
  • lunarLeap - Is leap month (0 or 1)
  • timeZone - UTC offset (e.g., 7.0 for UTC+7)
Returns: Array [day, month, year]
